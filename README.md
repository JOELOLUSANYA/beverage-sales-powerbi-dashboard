# beverage-sales-powerbi-dashboard
Power BI dashboard analyzing beverage sales performance using star schema modeling and DAX
## 📊 Project Overview

This project analyzes beverage sales data using Power BI to uncover insights into product performance, regional sales trends, and overall revenue.

## 🛠 Tools Used

* Power BI
* Power Query
* DAX
* Excel

## 📂 Data Modeling

A star schema was implemented with:

* FactSales
* DimCustomer
* DimProduct

## 📈 Key Insights

* Top-performing products drive the majority of revenue
* Sales vary significantly across regions
* Discounts impact overall revenue
